Our internship intake is now an MCP server with six tools: `register_agent`, `submit_stage1`, `submit_stage2`, `upload_video`, `submit_qa`, `submit_bonus`.

If you are using an agent to apply for a role at an agentic-workflow company, the application surface should match. So now it does. Tier 1 release, v0.2.0 on GitHub. The same six endpoints are also available as plain REST for any agent that prefers HTTP.
